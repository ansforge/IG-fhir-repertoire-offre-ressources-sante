ValueSet: Value Set type pour le contexte d'usage de la fiche de saisie (Experimental)

Official URL: https://interop.esante.gouv.fr/ig/fhir/ror/ValueSet/ror-usage-context-type-vs Version: 0.5.0-snapshot-1
Draft as of 2025-01-24 Computable Name: RORUsageContextTypeVS

Extension du Value Set http://hl7.org/fhir/ValueSet/usage-context-type pour fixer le type de contexte d’usage de la fiche de saisie avec le code EG ou le S pour Spécialité ordinale ou P01 pour le type de profession
