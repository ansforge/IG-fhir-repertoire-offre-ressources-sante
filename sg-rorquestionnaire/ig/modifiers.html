Paramètres et modificateurs de requêtes FHIR

 

Lien vers spécification

Description

Paramètres de modification du résultat d'une requête

_include

https://www.hl7.org/fhir/R4/search.html#include

Permet d'inclure des ressources référencées par la ressource recherchée dans le résultat (voir la documentation FHIR pour plus de précision)

_revinclude

https://www.hl7.org/fhir/R4/search.html#revinclude

Permet d'inclure les ressources qui font référence à la ressource recherchée dans le résultat (voir la documentation FHIR pour plus de précision)

_count

https://www.hl7.org/fhir/R4/search.html#count

Permet de choisir le nombre de résultat par page (voir la documentation FHIR pour plus de précision)

RG_RECH_52 : Par défaut la recherche retourne 200 résultats maximum par page.

_sort

https://www.hl7.org/fhir/R4/search.html#sort

Permet de choisir dans quel ordre renvoyer les résultats (voir la documentation FHIR pour plus de précision)

_elementsdraft

https://www.hl7.org/fhir/R4/search.html#elements

Permet au consommateur de demander les éléments à retourner de la ressource recherchée (voir la documentation FHIR pour plus de précision)

Modificateurs

 

:above

https://www.hl7.org/fhir/R4/search.html

Permet de faire des recherches hiérarchiques (voir la documentation FHIR pour plus de précision)

:iterate

https://www.hl7.org/fhir/R4/search.html

Permet de rendre le processus d'inclusion itératif (voir la documentation FHIR pour plus de précision)

:of-type

https://www.hl7.org/fhir/R4/search.html

Permet de filter sur le type d'identifiant (voir la documentation FHIR pour plus de précision)

:not

https://www.hl7.org/fhir/R4/search.html

Permet aux clients de filtrer si les ressources ne contiennent pas un élément spécifié en fonction de l'entrée du paramètre de recherche (voir la documentation FHIR pour plus de précision)

Préfixes de comparaison

 

gt, lt, ge, le, sa & eb

https://www.hl7.org/fhir/R4/search.html#prefix

Permet d’effectuer des recherches comparatives (voir la documentation FHIR pour plus de précision)
